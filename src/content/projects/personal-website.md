For this website, I built a text-first, typography-driven design with Astro and TypeScript. Every page is static HTML, generated at build time from markdown content and deployed to S3 through a GitHub Actions pipeline. A previous iteration of the site explored GSAP and Framer Motion animations; the current design trades spectacle for clarity, fast loads, and readable prose.
